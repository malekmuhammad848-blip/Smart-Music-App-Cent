# CENT Music Streaming Platform

## Prerequisites
- Docker & Docker Compose
- Node.js 18+
- Flutter 3.0+
- Python 3.11+

## Quick Start

1. Clone the repository
2. Run: `docker-compose up -d`
3. Backend: http://localhost:3000
4. Frontend: http://localhost:8080
